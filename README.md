# Library Explorer

A React & TypeScript application for browsing books with vintage aesthetic.
This project was built as a home assignment to demonstrate frontend development skills.

**Click here to view the live project:** (https://library-explorer-theta.vercel.app/)

##  Features

* **Search:** Real-time filtering by book title or author.
* **Advanced Filtering:**
    * Filter by Category (Tech, Fiction, History, etc.).
    * Filter by Minimum Rating (Interactive star rating system).
* **Sorting:** Sort books by Title (A-Z / Z-A) or Rating (Low-High / High-Low).
* **Favorites System:** Mark books as favorites with persistence using LocalStorage (data remains after refresh).

## Tech Stack

* **Framework:** React (Hooks: `useState`, `useEffect`)
* **Language:** TypeScript
* **Styling:** CSS
* **Data:** Local JSON file simulation

##  Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    bash
    git clone https://github.com/sashabel1/Library-Explorer
    
2.  **Install dependencies:**
    bash
    npm install
    
3.  **Run the app:**
    bash
    npm start
